# Gum

Deploy production functions on Gum.

Gum is a managed runtime for background jobs. The first version is focused on
Python execution with:

- enqueue
- schedules
- retries
- timeouts
- rate limits
- concurrency
- deduplication keys
- logs
- replay

Locked product and architecture docs live in [`docs/`](./docs).

The marketing site now lives in [`website/`](./website).
