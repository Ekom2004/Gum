# Gum

Run Python functions in the cloud.

Gum is a Python function cloud for reliable background work. The first version
is focused on Python execution with:

- enqueue
- schedules
- retries
- timeouts
- memory sizing
- rate limits
- concurrency
- deduplication keys
- logs
- replay

Locked product and architecture docs live in [`docs/`](./docs).

The marketing site now lives in [`website/`](./website).
